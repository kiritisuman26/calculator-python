# Calculator (Python)

A simple console-based calculator built using Python. This project performs basic arithmetic operations through a menu-driven interface.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus
* Power
* Division by zero handling
* Modulus by zero handling
* Menu-driven interface
* Repeated calculations until user exits

## Concepts Used

* Python Functions
* Loops (`while`)
* Conditional Statements (`if-elif-else`)
* User Input
* Input Validation
* Return Values

## How to Run

1. Clone the repository.
2. Navigate to the project folder.
3. Run the program:

```bash
python calculator.py
```

## Sample Menu

```text
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Modulus
6. Power
7. Exit
```
